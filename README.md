# Анализ временных рядов — Учебно-методический комплекс

Полный учебно-методический комплекс (УМК) для освоения основ анализа временных рядов с использованием Python. Проект включает 16 интерактивных Jupyter notebooks с теоретическим материалом, практическими примерами и заданиями разных уровней сложности.

## 1. Описание проекта

**Краткое название:** Разработка УМК "Анализ временных рядов"

**Тип проекта:** Прикладной (практико-ориентированный)

**Цель:** Развитие навыков программирования и знакомство с методами анализа и прогнозирования временных рядов.

**Требуемый результат:** Набор Jupyter notebooks с теорией, примерами кода и практическими заданиями.

---

## 2. Критерии завершения проекта

- 16 Jupyter notebooks с теоретическим материалом и заданиями
- Единый стиль оформления (визуализация, структура, оформление кода)
- Техническая документация (README, требования к окружению, инструкции по запуску)
- Интерактивное обучение (примеры кода и практические задания)

## 5. Установка и запуск

### Шаг 1: Клонирование репозитория

```bash
git clone https://github.com/graiustommy/TimeSeriesAnalysis.git
cd TimeSeriesAnalysis
``````
### Шаг 2: Создание виртуального окружения

Для Linux/macOS:
```bash
python3 -m venv venv
source venv/bin/activate
```

Для Windows:
```bash
python -m venv venv
venv\Scripts\activate
```

### Шаг 3: Установка зависимостей

```bash
pip install --upgrade pip
pip install -r requirements.txt
```

### Шаг 4: Запуск Jupyter

Для классического Jupyter:
```bash
jupyter notebook
```

Для JupyterLab:
```bash
jupyter lab
```

### Шаг 5: Открытие notebook

После запуска Jupyter откройте интересующий вас notebook из папки `notebooks/`.

## 14. Часто задаваемые вопросы

**В: Какой опыт программирования требуется для прохождения курса?**

О: Рекомендуется базовое знание Python (переменные, функции, циклы, работа с функциями). Если вы не знакомы с Python, рекомендуется предварительно пройти вводный курс.

**В: Сколько времени займет прохождение всего курса?**

О: В зависимости от глубины погружения и выполнения заданий потребуется 80-180 часов. Для быстрого ознакомления можно потратить 30-40 часов на основные modules.

**В: Могу ли я использовать материал для обучения студентов?**

О: Да. Материал распространяется под лицензией CC BY 4.0, которая позволяет образовательное использование при указании источника.

**В: Есть ли готовые решения к заданиям?**

О: Решения доступны в ветке `solutions` репозитория. Данная ветка создана для преподавателей и может быть недоступна публично.

**В: Какие датасеты используются в курсе?**

О: Используются синтетические данные (сгенерированные для демонстрации) и публичные датасеты из источников: Kaggle, UCI Machine Learning Repository, Yahoo Finance и других открытых архивов.

**В: Совместима ли система с Windows/Mac/Linux?**

О: Да, все инструменты кроссплатформенны. Инструкции учитывают различия между операционными системами.

**В: Какая версия Python рекомендуется?**

О: Python 3.8 и выше. Рекомендуется использовать последнюю стабильную версию (3.10+).
